## Features

イベントを追加し、予定を入れることができます。
 

 
## Installation

 
```bash
git clone https://github.com/shimish000/calendartodo.git

cd /calendartodo
```
 
## Usage
ログインしていただくと予定が追加できる仕組みとなっております。右上のボタンからログイン後、ボタンがnew eventに変わるので、そこからイベントを追加してください。
